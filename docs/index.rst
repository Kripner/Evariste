Evariste
========

Codebase overview
-----------------

The codebase implements the following components:

* Vanilla transformer implementation
* Pretraining (e.g. MLM, CLM, MASS tasks) and supervised training (e.g. seq2seq task)
* Implementation of 5 mathematical formal systems (Equations (debug environment), Metamath, HOL-light, Lean, Isabelle)
* Hypertree proof search theorem provers
* Hypertree proof search online training
